## 3. 命名法总观

有机化学命名的目的不是“给化合物起个叫法”，而是用书面或口头名称唯一指认化学物种，并让名称显式或隐式对应结构。

核心名称类型：

| 类型 | 含义 | 复习重点 |
|---|---|---|
| Preferred IUPAC name, PIN | BlueBook 推荐的优先名称 | 命名题优先采用 |
| General IUPAC name | 一般 IUPAC 可接受名称 | 可用于普通语境，但不一定是 PIN |
| Preselected name | 对某些母体结构预先选定的名称 | 常见于无机/杂原子母体 |
| Retained name | 保留传统名 | 如 benzene, pyridine, acetic acid 等；是否可取代要看规则 |

课件涉及的 8 类命名操作：

| 操作 | 英文 | 作用 |
|---|---|---|
| 取代 | substitutive | 以母体氢化物为基础，用前缀/后缀表示取代 |
| 置换 | replacement | 用 `a` 前缀表示骨架原子被杂原子替代，如 aza-, oxa-, thia- |
| 加合 | additive | 表示加成或组合 |
| 减脱 | subtractive | 如 deoxy- 等表示少了原子/基团 |
| 缀合 | conjunctive | 将两个命名单元连接 |
| 复合 | multiplicative | 多个相同母体由多价基连接，如 methylenedibenzonitrile |
| 并合 | fusion | 稠环命名 |
| 天然产物专用操作 | natural product operations | 天然产物命名中的特殊改造 |

## 4. 取代命名法主公式

BlueBook 与课件共同主线：

```text
prefixes -- parent -- endings -- suffixes
前缀 -- 母体 -- 不饱和词尾 -- 官能团后缀
```

展开后：

| 名称部分 | 组成 | 功能 |
|---|---|---|
| Prefixes | 不可分离结构前缀、可分离饱和/不饱和前缀、可分离取代前缀 | 说明环、稠合、杂原子、取代基等 |
| Parent | parent hydride 或 functional parent compound | 名称骨架 |
| Endings | -ane, -ene, -yne | 表示饱和、不饱和 |
| Suffixes | functional suffixes, cumulative suffixes | 表示主官能团、离子/自由基等 |

名称书写符号：

| 符号 | 用途 | 示例 |
|---|---|---|
| 逗号 | 分隔多个位次 | 5,8-tetramethyl |
| 连字符 | 连接位次、构型、前缀和主体 | non-4-enamide |
| 括号 | 包住复杂取代基或构型 | (4E)- |
| 空格 | 功能类别命名中分隔词 | acetyl chloride |
| N, O 等斜体元素符号 | 表示取代发生在杂原子上 | N,N-dimethyl |

## 6. 环烷烃、芳烃与环系

### 环烷烃

饱和单环烃用不可分离前缀 `cyclo-` 加同碳数非环烷烃名：

```text
cyclo + alkane -> cycloalkane
```

例：`1-methylcyclopentane`。

编号原则：给取代基最低位次；若位次集合相同，按名称中先引用的取代基给较低位次。  
例：`3-ethyl-5-methylheptane`，不是 `5-ethyl-3-methylheptane`。

### 芳烃

芳香化合物相对氢少，常见母体与保留名：

| 名称 | 说明 |
|---|---|
| benzene | PIN，可作取代母体 |
| toluene | PIN，但课件标注不再作为进一步取代母体使用 |
| xylene | 通用名，不是优先系统名 |
| phenol | PIN，可取代 |
| aniline | PIN，可在环和氮上取代 |
| benzaldehyde | PIN |
| benzoic acid | PIN |

二取代苯常见传统定位词：

| 传统词 | 位次 | 中文 |
|---|---|---|
| ortho-, o- | 1,2- | 邻 |
| meta-, m- | 1,3- | 间 |
| para-, p- | 1,4- | 对 |

BlueBook 取向：PIN 中优先使用数字位次，而不是 o-/m-/p-。

### 联环与稠环

两个或多个相同环直接由单键或双键连接，称 ring assemblies。常见倍数前缀：

| 数目 | 前缀 |
|---|---|
| 2 | bi- |
| 3 | ter- |
| 4 | quater- |
| 5 | quinque- |
| 6 | sexi- |

例：`1,1'-biphenyl`，`2,2'-bipyridine`。

## 7. 杂环与 Hantzsch-Widman 命名

杂环命名主线：

```text
heteroatom prefix + ring-size/root + saturation suffix
杂原子前缀 + 环大小词根 + 饱和/不饱和后缀
```

常用杂原子前缀：

| 杂原子 | 前缀 |
|---|---|
| N | aza- |
| O | oxa- |
| S | thia- |
| P | phospha- |

常用环大小后缀：

| 环大小 | 不饱和 | 饱和 |
|---|---|---|
| 3 | -irene / -irine | -irane / -iridine |
| 4 | -ete | -etane / -etidine |
| 5 | -ole | -olane / -olidine |
| 6 | -ine | -inane / -ane |
| 7 | -epine | -epane |
| 8 | -ocine | -ocane |
| 9 | -onine | -onane |
| 10 | -ecine | -ecane |

关键规则：

1. 单一杂原子通常定位为 1 位。
2. 多个同类杂原子用 di-, tri-, tetra- 等。
3. `tetrazole` 中 `tetra-` 的末尾 `a` 在元音前省略，不能写成 `tetraazole`。
4. 多种杂原子并存时，引用顺序为：F, Cl, Br, I, O, S, Se, Te, N, P, As, Sb, Bi, Si, Ge, Sn, Pb, B, Al, Ga, In, Tl。
5. 编号先给引用顺序更高的杂原子最低位次，再给杂原子集最低位次组。

## 8. 不饱和词尾与取代基

### -ene / -yne

双键和三键通过把饱和母体的 `-ane` 改为 `-ene` 或 `-yne` 表示。

规则：

1. 多重键作为一个集合获得尽可能低的位次。
2. 若仍有选择，双键优先获得较低位次。
3. 名称中 `ene` 总在 `yne` 前。
4. locant 放在它修饰的名称部分之前。
5. 几何异构用 cis/trans 或 E/Z 标注。
6. `ene` 后接元音时可省略末尾 `e`：`pent-3-en-1-yne`，不是 `pent-3-ene-1-yne`。
7. 加 di-, tri- 等倍数前缀时，为避免两个辅音相连，可插入 `a`：`buta-1,3-diene`。

保留名：

| 保留名 | PIN 或说明 |
|---|---|
| acetylene | PIN |
| allene | PIN: propa-1,2-diene |
| isoprene | PIN: 2-methylbuta-1,3-diene |

### 取代基后缀

由母体失去氢产生游离价，用以下后缀：

| 游离价类型 | 后缀 |
|---|---|
| 一价 | -yl |
| 二价，来自不同原子 | -diyl |
| 二价，来自同一原子形成双键连接 | -ylidene |
| 三价，来自同一原子 | -ylidyne |
| 三价，来自不同原子 | -triyl |

例：

| 结构/含义 | 优先前缀 |
|---|---|
| CH3CH2- | ethyl |
| CH3CH2CH2- | propyl |
| (CH3)2CH- | propan-2-yl |
| CH2= | methylidene |
| propane-1,3-diyl | 不是 trimethylene |

易错点：`isopropyl`, `isobutyl`, `sec-butyl`, `neopentyl` 等保留前缀常见，但不总是 PIN；系统命名时应优先使用 `propan-2-yl`, `2-methylpropyl`, `butan-2-yl`, `2,2-dimethylpropyl` 等。

## 9. 官能团与优先级

多官能团化合物只能选择一个 principal characteristic group 作为后缀，其余作为前缀。

课件给出的优先级核心序列：

1. 自由基
2. 自由基阴离子
3. 自由基阳离子
4. 阴离子
5. 两性离子
6. 阳离子
7. 酸类：羧酸、过氧酸及 S/Se 衍生物，随后为磺酸、亚磺酸等
8. 酸衍生物：酸酐、酯、酰卤/拟卤化物、酰胺、酰肼、酰亚胺、腈
9. 醛及其类似物和衍生物
10. 酮及其类似物和衍生物
11. 醇和酚，及 S/Se/Te 类似物
12. 氢过氧化物
13. 胺
14. 亚胺、肼等

常见官能团表：

| 类别 | 结构 | 前缀 | 后缀/类别名 | 中文 |
|---|---|---|---|---|
| 卤代烃 | R-X | fluoro-, chloro-, bromo-, iodo- | 无后缀 | 卤代烃 |
| 醚 | R-O-R' | alkyloxy- / alkoxy- | ether 一般名 | 醚 |
| 醇 | R-OH | hydroxy- | -ol | 醇 |
| 酚 | Ar-OH | hydroxy- | -ol / phenol | 酚 |
| 胺 | R-NH2 | amino- | -amine | 胺 |
| 亚胺 | R2C=NH | imino- 等 | -imine | 亚胺 |
| 醛 | R-CHO | formyl- / oxo- | -al / -carbaldehyde | 醛 |
| 酮 | R-CO-R' | oxo- | -one | 酮 |
| 羧酸 | R-COOH | carboxy- | -oic acid / -carboxylic acid | 羧酸 |
| 酰胺 | R-CONR2 | carbamoyl- | -amide / -carboxamide | 酰胺 |
| 腈 | R-CN | cyano- | -nitrile / -carbonitrile | 腈 |
| 酰卤 | R-CO-X | halocarbonyl- | acyl halide 类别名 | 酰卤 |
| 酯 | R-COOR' | alkoxycarbonyl- | -ate / -carboxylate | 酯 |
| 酸酐 | R-CO-O-CO-R' | oxoacyloxy- | anhydride 类别名 | 酸酐 |

## 10. 各类化合物命名要点

### 卤代物

PIN 用取代命名法，卤素只作前缀：

```text
fluoro-, chloro-, bromo-, iodo-
```

例：`chloroethane`, `1,2-difluoropropane`, `2-bromo-2-methylpropane`, `iodomethane`。  
`fluoroform`, `chloroform`, `bromoform`, `iodoform` 是保留名，但不是优先名。

### 醚

PIN 中把 `R-O-` 作为 `alkyloxy-` 前缀接到母体上。  
常见保留优先前缀：`methoxy-`, `ethoxy-`, `propoxy-`, `butoxy-`, `phenoxy-`, `tert-butoxy-`。

环醚：小环多用 Hantzsch-Widman；十元以上可用骨架置换命名。

### 醇与酚

醇用功能后缀 `-ol`；若母体末尾有 `e`，接 `-ol` 时省略。编号时给 `-ol` 最低位次。

例：`propan-2-ol`, `ethane-1,2-diol`, `propane-1,2,3-triol`。  
`phenol` 是保留 PIN，可全位置取代；PIN 推荐使用 2/3/4 位次而非 o/m/p。

### 胺与亚胺

胺优先使用取代命名法和后缀 `-amine`。  
`aniline` 是保留 PIN，可在环和 N 上取代；`anilino-` 是保留优先前缀。  
季铵盐 R4N+X-：由胺名加 `-ium`，省略末尾 `e`。  
亚胺使用后缀 `-imine`。

### 醛与酮

醛：

| 结构 | 后缀 |
|---|---|
| -(C)HO，碳在主链内 | -al |
| -CHO，连在环或母体外 | -carbaldehyde |

酮：用 `-one` 作后缀，`oxo-` 作前缀。  
多酮可用 `dione`, `trione` 等；如 `tetrone` 中倍数词末尾 `a` 在 `one` 前省略。

### 羧酸及衍生物

羧酸：

| 结构 | 后缀 |
|---|---|
| -(C)OOH，碳计入主链 | -oic acid |
| -COOH，连在环或母体外 | -carboxylic acid |

酰胺：

| 结构 | 后缀 |
|---|---|
| -CONH2 接在链上，羰基碳计入主链 | -amide |
| -CONH2 接在环/杂原子上，羰基碳不计入母体 | -carboxamide |

腈：

| 结构 | 后缀 |
|---|---|
| -(C)N，腈碳计入主链 | -nitrile |
| -CN，腈碳不计入母体 | -carbonitrile |

酰卤使用功能类别命名：

```text
acyl group + halide
```

例：`acetyl chloride`, `propanoyl bromide`, `cyclohexanecarbonyl bromide`。

酯和盐：

```text
cation/organyl + acid-derived anion ending in -ate
```

例：`ethyl ethanoate`, `ethyl cyclohexanecarboxylate`。

酸酐：

| 类型 | 命名 |
|---|---|
| 对称无环酸酐 | 将 acid 改为 anhydride |
| 混合酸酐 | 两个酸名去掉 acid，按字母顺序 + anhydride |
| 环状酸酐 | PIN 常作为杂环拟酮命名 |

内酯：羟基羧酸的分子内酯，PIN 常按杂环 pseudoketone 用 `-one`, `-dione`, `-thione` 等命名。  
内酰胺/内酰亚胺：对应 lactam/lactim 体系。

## 11. 多官能团命名流程

做命名题时按下面顺序：

1. 找全部官能团、取代基、环、杂原子和不饱和键。
2. 按 BlueBook/课件优先级选择 principal characteristic group。
3. 选择母体：
   - 优先包含主官能团。
   - 在同类选择中，环通常优先于链。
   - 链长、重键数、取代数、最低位次依次比较。
4. 给母体编号：
   - 主官能团最低位次。
   - 多重键最低位次。
   - 取代基最低位次组。
5. 把非主官能团改写为前缀，如 hydroxy-, amino-, oxo-, cyano-, formyl-, carboxy-。
6. 前缀按名称排序，复杂取代基加括号。
7. 写出完整名称：

```text
(stereochemistry)-locants-prefixes-parent-unsaturation-functional suffix
```

8. 检查书写：逗号、连字符、括号、空格、N/O 位取代、E/Z、R/S、元音省略。

课件例题：

| 结构特征 | 名称 |
|---|---|
| 苯甲酸为主官能团，另有 formyl 和 nitro | 3-formyl-4-nitrobenzoic acid |
| 酯为高优先级酸衍生物，另有 bromo/cyano/formyl | ethyl 7-bromo-6-cyano-5-formylheptanoate |
| 环己烷羧酸为母体，另有 cyano/oxo | 2-cyano-4-oxocyclohexanecarboxylic acid |
| 酰胺为主官能团，另有 amino/hydroxy/methyl | 7-amino-3-hydroxy-6-methyloctanamide |

## 13. 最小复习清单

考前优先掌握以下内容：

1. `prefixes--parent--endings--suffixes` 的名称结构。
2. PIN、retained name、general name 的区别。
3. 烷烃 1-12、20、21、30 的英文和烷基形式。
4. `cyclo-`、benzene/phenol/aniline/benzoic acid 等保留母体。
5. Hantzsch-Widman：aza-, oxa-, thia- 与 -irane/-olane/-ine 等。
6. `-ene/-yne` 位次、E/Z、元音省略规则。
7. `-yl/-diyl/-ylidene/-ylidyne` 的含义。
8. 官能团前缀/后缀表，尤其是酸及酸衍生物、醛、酮、醇、胺。
9. 多官能团优先级：酸 > 酸衍生物 > 醛 > 酮 > 醇/酚 > 胺。
10. 名称书写符号：逗号、连字符、括号、空格、N/O 位取代。
